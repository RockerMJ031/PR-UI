Purple Ruler - School Dashboard

Project Overview

Purple Ruler is a comprehensive school management system designed to streamline administrative tasks, enhance student tracking, and improve communication between mentors, students, and administrators. This dashboard provides an intuitive interface for managing all aspects of school operations.

Key Features

Student Management

Comprehensive student profiles with academic history, attendance records, and performance metrics.

Session Scheduling

Intuitive calendar interface for scheduling and managing classes, appointments, and events.

Performance Analytics

Detailed reports and analytics to track student progress and identify areas for improvement.

Financial Management

Tools for managing tuition, fees, payments, and generating financial reports.

Technical Features

  • Responsive design that works on desktop, tablet, and mobile devices
  • Modern UI with intuitive navigation and clean visual hierarchy
  • Interactive components with real-time feedback
  • Data visualization for better understanding of student performance
  • Role-based access control for administrators, mentors, and students

Tech Stack

  • HTML5, CSS3, and JavaScript for frontend development
  • Responsive design using CSS Grid and Flexbox
  • Font Awesome for icons
  • Custom CSS variables for consistent theming

Getting Started

To get started with Purple Ruler, simply navigate through the sidebar menu to access different modules of the system. Each page is designed to provide specific functionality related to its domain.

  • Dashboard: Overview of key metrics and recent activities
  • Students: Manage student profiles and track their progress
  • Sessions: Schedule and manage classes and appointments
  • Reports: Generate and view various reports and analytics
  • Finance: Manage financial aspects including payments and invoices

Deployment

Purple Ruler is a static HTML/CSS/JS application that can be deployed on any web server. Simply upload the files to your server and access the index.html file to start using the system.

Design Principles

  • Simplicity: Clean and intuitive interface that's easy to navigate
  • Consistency: Uniform design language across all pages
  • Accessibility: Designed to be accessible to all users
  • Responsiveness: Works seamlessly on all device sizes
