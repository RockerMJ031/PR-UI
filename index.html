Purple Ruler Academy - Mentor Portal

A comprehensive educational management system for mentors and administrators

Project Overview

Purple Ruler Academy Mentor Portal is a modern, responsive web application designed to streamline educational management processes. This comprehensive system provides mentors and administrators with powerful tools to manage students, sessions, reports, and financial operations efficiently.

Key Features

Student Management

Comprehensive student enrollment, tracking, and management system with detailed profiles and progress monitoring.

Session Scheduling

Advanced calendar-based session management with attendance tracking and flexible scheduling options.

Reporting System

Detailed analytics and reporting tools for student progress, attendance, and safeguarding requirements.

Financial Management

Complete financial oversight with pricing plans, invoice management, and payment tracking.

Technical Features

  • Responsive Design: Fully optimized for desktop, tablet, and mobile devices
  • Modern UI/UX: Clean, intuitive interface with consistent design patterns
  • Interactive Components: Dynamic forms, modals, and real-time updates
  • Accessibility: WCAG compliant with keyboard navigation support
  • Performance Optimized: Fast loading times and smooth interactions

Technology Stack

HTML5 CSS3 JavaScript Font Awesome Responsive Design

Project Structure

  • index.html - Main dashboard and project overview
  • students-enhanced.html - Student management interface
  • sessions-enhanced.html - Session scheduling and management
  • reports-enhanced.html - Reporting and analytics dashboard
  • finance.html - Financial management and pricing

Getting Started

To run this project locally:

  1. Clone or download the project files
  2. Open index.html in your web browser
  3. Navigate through the different sections using the sidebar menu

Deployment

This project is ready for deployment on:

  • GitHub Pages: Simply push to a GitHub repository and enable Pages
  • Netlify: Drag and drop the folder for instant deployment
  • Vercel: Connect your repository for automatic deployments
  • Any web server: Upload files to your hosting provider

Design Principles

  • User-Centered: Designed with mentor and administrator workflows in mind
  • Consistent: Unified color scheme and component styling throughout
  • Accessible: Proper contrast ratios and semantic HTML structure
  • Scalable: Modular CSS architecture for easy customization
